# Athrye - Numismatic Marketplace

A modern platform for collectors to discover, buy and sell authentic coins built with cutting-edge web technologies.

## Architecture

The project architecture alignment report and phased migration roadmap live in [docs/architecture-alignment-gap-report.md](docs/architecture-alignment-gap-report.md).

## ⚡ Tech Stack

### Frontend Framework

- **Next.js 16** - App Router
- **React 19** - Server Components
- **TypeScript** - Type Safety

### Styling & Animation

- **Tailwind CSS** - Utility-first CSS
- **Motion** - Interactive animations
- **Shadcn/UI** - Reusable components
- **Lucide Icons** - Icon library

### Development Tools

- **ESLint** - Linting
- **Prettier** - Code formatting
- **PostCSS** - CSS processing

## 🚀 Features

### Design & UX

- Responsive mobile-first design
- Dark/light mode with system preference sync
- Smooth page transitions and animations
- Modern minimalist interface
- Custom font implementation (Montserrat & Playfair Display)

### Core Features

- Interactive product catalog
- Advanced search functionality
- Product filtering
- User authentication ready
- Shopping cart system
- Responsive image grid gallery

### Technical Features

- Server-side rendering (SSR)
- Client-side animations
- Type-safe development
- Optimized performance
- Responsive images
- Clean component architecture
